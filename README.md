## Description
Space Invaders like made with Qt 6.9 in C++ & QML.
![Gameplay image](https://github.com/FoxBandyKoot/Space-Aliens/blob/Com/Com/Gameplay.jpg)
## Installation
1. Open `CMakeLists.txt` in Qt Creator to load the project.
2. Build to run.
